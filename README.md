# Remix CRUD App

Full CRUD task manager using Remix 2 resource routes with loader/action, in-memory data layer, and ErrorBoundary on every route.

**Stack:** Remix | **Complexity:** moderate
**Tags:** `remix`, `react`, `crud`, `tasks`, `forms`, `actions`, `loaders`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
